# php 扩展加载
## 1  PHP_MINIT 宏
- 首先又宏替换
`PHP_MINIT(moule)`  等价于   `zm_startup_moule`

具体的宏替换如下 

```
#define PHP_MINIT		ZEND_MODULE_STARTUP_N

#define ZEND_MODULE_STARTUP_N(module)       zm_startup_##module
```
